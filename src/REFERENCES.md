## API Coverage
- Calendars: list, get, create, delete
- Events: list, get, create, update, delete, move
- Quick Add: natural language event creation
- Attendees: add, remove, respond to event
- Recurring Events: list instances
- Free/Busy: check availability across calendars
- Settings: get calendar settings
